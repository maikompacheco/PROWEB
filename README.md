# PROWEB Sports — Plataforma de Gestão Esportiva

Scaffold inicial: React 18 + TypeScript + Vite + TailwindCSS.

Comandos:

```bash
npm install
npm run dev
# build
npm run build
npm run preview
```

Arquivos importantes:
- [index.html](index.html)
- [src/main.tsx](src/main.tsx)
- [src/App.tsx](src/App.tsx)
- [tailwind.config.cjs](tailwind.config.cjs)

Descrição rápida: scaffold para gerenciar atletas e equipes escolares, com páginas para listagem e tracking.
